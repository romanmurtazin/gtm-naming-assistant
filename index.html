What it does

Що вміє

Everything you need for
consistent GTM naming

Все для єдиного
іменування в GTM

A suite of smart tools that automate the repetitive parts of working with GTM naming conventions.

Набір розумних інструментів, які автоматизують рутинну роботу з конвенціями іменування в GTM.

🏷️

Auto-prefix

Select a tag, trigger, or variable type — the name field is filled with your prefix instantly. Customizable for every type. Works with English, Ukrainian, and Russian GTM interfaces.

Автопрефікс

Оберіть тип тега, тригера або змінної — поле назви миттєво заповнюється вашим префіксом. Налаштовується для кожного типу. Працює з англійським, українським та російським інтерфейсом GTM.

⚡

Auto-append trigger name

Assign a trigger to a tag — the trigger name is automatically appended to the tag name. Change the trigger, name updates too. Works correctly on copied tags.

Автододавання назви тригера

Призначте тригер до тега — назва тригера автоматично додається до назви тега. Зміните тригер — назва оновиться. Коректно працює на скопійованих тегах.

📊

GA4 Event Name Grabber

Converts the trigger name to snake_case and fills the Event Name field in GA4 Event tags automatically. Also works when creating tags via Copy.

GA4 Event Name Grabber

Конвертує назву тригера у snake_case і автоматично заповнює поле Event Name у тегах GA4 Event. Також працює при створенні тегів через Copy.

📦

DLV Helper

Type the Data Layer Variable Name — the variable title updates in real time. No more copy-pasting between fields.

DLV Helper

Введіть Data Layer Variable Name — назва змінної оновлюється в реальному часі. Більше ніякого копіювання між полями.

🔖

Tag Suffix New

Automatically append a custom suffix (e.g. "- APR-2026") to GA4 and Google Ads tags. Works together with auto-append trigger name.

Tag Suffix Новинка

Автоматично додає кастомний суфікс (напр. "- APR-2026") до тегів GA4 та Google Ads. Працює разом із автододаванням назви тригера.

📌

Sticky sidebar New

Pin the left navigation menu so it stays visible while scrolling through long lists of tags, triggers, variables, folders, and templates.

Sticky sidebar Новинка

Закріплює ліве навігаційне меню, щоб воно залишалося видимим під час прокручування довгих списків тегів, тригерів, змінних, папок і шаблонів.

📋

One-click copy

Hover over any tag, trigger, or variable in the list — a copy button appears. Click to copy the name to clipboard instantly.

Копіювання в один клік

Наведіть курсор на будь-який тег, тригер або змінну у списку — з'явиться кнопка копіювання. Клік — і назва вже в буфері обміну.

🎨

Tag type icons

Optionally display recognizable icons next to tag types in the list for faster visual scanning.

Іконки типів тегів

За бажанням відображає іконки поруч із типами тегів у списку для швидшого візуального сканування.
